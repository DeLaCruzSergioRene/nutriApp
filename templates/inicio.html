Este proyecto analiza tres aplicaciones de nutrición: MyFitnessPal, FatSecret y Cronometer.

Se evaluaron sus principales funciones, experiencia de usuario y modelo de negocio para identificar fortalezas y debilidades.

Objetivos

  • Analizar características clave de las aplicaciones de nutrición.
  • Conocer las preferencias de los usuarios mediante una encuesta.
  • Proponer mejoras para una futura app nutricional.
