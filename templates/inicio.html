Proyecto de Aplicación Nutricional

Este proyecto analiza tres aplicaciones populares de nutrición: MyFitnessPal, FatSecret y Cronometer.

Introducción

Se evaluaron sus principales funciones, la experiencia de usuario y el modelo de negocio para identificar sus fortalezas y debilidades. Con base en esta información se busca proponer una futura aplicación que combine lo mejor de cada una.

Objetivos

  • Analizar las características clave de las aplicaciones de nutrición.
  • Conocer las preferencias de los usuarios mediante una encuesta.
  • Detectar oportunidades de mejora para un futuro desarrollo.
